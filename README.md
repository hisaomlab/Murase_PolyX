# Repository Overview
This folder contains the code and datasets used in the paper: 
Systematic overexpression reveals a conserved spectrum of harmful and beneficial amino acid homorepeats shaped by evolution.

DOI: 

# Contents
Code/
Scripts used for the image analyses performed in the paper, including:

Cell size measurements

Cell axis ratio calculations

Expression analysis of RPN4 and genes regulated HSF1

Raw_data/

Data for figure generation (graphs) and raw gel images. including:

RNA-seq datasets used for the analyses

Proteome datasets used for the analyses

Data for figure generation (graphs shown in the paper)

Raw gel data corresponding to each figure

Omics_analysis/

Datasets and scripts required for the RNA-seq ().

# Usage
The scripts can be opened and executed by python3 in VScode to reproduce the figures and analyses presented in the paper.
